A web browser or browser ( English : web browser or browser , cultural language : browser ) is an application software based on a graphical user interface that navigates from a web server , communicates in two directions, and outputs HTML documents or files . The web browser is also one of the representative HTTP user agents . Major web browsers include Mozilla Firefox , Google Chrome , Internet Explorer / Microsoft Edge , [1] [2] [3] Opera , and Safari .
